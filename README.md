# Cypress

Curso Cypress

### Intalando o cypress

-npm i cypress@9.1.1 --save -include=dev

### Comando

- npm run test

### O modelo Triple A é um padrão que diz que todo o teste unitário deve possuir três etapas:

- Arrange (Preparar o teste)
- Act (Rodar o teste)
- Assert (Verificar as asserções).
